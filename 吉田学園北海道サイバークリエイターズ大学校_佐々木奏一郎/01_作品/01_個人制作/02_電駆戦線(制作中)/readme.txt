◆操作方法◆
キャラクター操作
Qキー : 左タイヤが前に進む
Aキー : 左タイヤが後ろに進む
Eキー : 右タイヤが前に進む
Dキー : 右タイヤが後ろに進む
SPACEキー : レールに乗る/離れる

デバッグ
F1 : デバッグ表示

ESC : ゲームを閉じる
