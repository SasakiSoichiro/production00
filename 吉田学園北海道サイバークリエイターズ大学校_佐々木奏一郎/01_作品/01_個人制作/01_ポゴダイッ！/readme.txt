◆作品について◆

◇使用言語 : C言語
◇SDK      : DirectX9
◇開発期間 : 2024/10～2024/11(一か月)
◇ジャンル : アクションゲーム

◆動作確認済み環境◆
Windows11
（DirectxInput対応のゲームパッド(ジョイスティック)に対応しています。）

◆操作方法◆
<<キーボード>>
Q/E : 左回転/右回転
F : リトライ
W/A/S/D : ステージ選択(上/左/下/右)
ENTER : 決定・進む・アイテム設置
ESC : 終了

<<コントローラー>>
左スティック・十字ボタン : 左回転/右回転
十字ボタン : ステージ選択
Xボタン : 設置
Aボタン : ジャンプ(条件あり)・決定
LB＋RB : リトライ
