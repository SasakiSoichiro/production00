◆作品について◆

◇使用言語 : C++
◇SDK      : DirectX9
◇開発期間 : 2025/06～09
◇ジャンル : レールアクションゲーム
◆動作確認済み環境◆
Windows11

◆操作方法◆
<<キーボード>>
Q : 左タイヤが前に進む
A : 左タイヤが後ろに進む
E : 右タイヤが前に進む
D : 右タイヤが後ろに進む
S : 後ろ回避
SPACE : レールに乗る/離れる
L/RSHIFT : ジャンプなしで坂を登ります
ENTER : 決定・進む
ESC : 終了

<<コントローラー>>※XINPUTのコントローラーで動作します。
左スティック : 前/後ろ
LB/RB : 左/右に曲がる
X/A : 回転
Y : ジャンプなし/レールに乗る/離れる
B : 後ろ回避
A : 決定
START : ゲーム開始

◆ゲームの進み方◆
・制限時間内または充電切れになる前に敵を倒す
・制限時間2分　合計3ウェーブ

◆敵の種類/攻撃方法/倒し方◆

蜘蛛型
攻撃方法 : 放電して攻撃(充電を増やす/100%超えるとスタン)
倒し方 : 重なって一回転することで倒せる

人型 
攻撃方法 : 腕のチェンソーを振って攻撃(充電を減らす)
倒し方 : 床に落ちているねじの上で回転することでばらまいてぶつけて倒せる

◆ゲームの仕様◆
レールに乗ると比較的早く(充電/放電)することができる
空中で回転することで少しずつ充電することができる
スタンするたびに充電の消耗速度が速くなる
人型敵の攻撃をタイミングよく回避するとスローになる
着地時に回転ボタンを押していると曲がる
レール中に乗り換えできる
